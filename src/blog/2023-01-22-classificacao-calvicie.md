Existe uma classificação para a calvície, que vai até 7 graus para os homens e até 3 para as mulheres, pois as características da calvície feminina são diferentes.

Mas, independente do grau que cada paciente apresente, nós podemos realizar uma megasessão de transplante capilar.

Para isso, caso não tenhamos uma boa área doadora, podemos contar com pelos corporais para complementar a quantidade de folículos necessária para um bom resultado cirúrgico.

Portanto, se você tem um grau avançado de calvície e tem dúvidas se seria possível realizar um transplante capilar, agende a sua consulta e venha saber mais.
